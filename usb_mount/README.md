Author

Description
===
Easily mounts USB partitions when using window managers as opposed to desktop environments

Usage
===
`usb_mount [-h] USB_PARTITION [MOUNT_POINT]`

* h: Help

	Displays the help message

USB_PARTITION

* USB mount partition

MOUNT_POINT

* Optional mount point
